# Penguins Data Analysis

## Project Overview

This project analyzes the Palmer Penguins dataset using Microsoft Excel. Exploratory data analysis (EDA), descriptive statistics, correlation analysis, boxplots, heatmaps, and linear regression were performed to identify relationships between penguin characteristics.

## Analysis Performed

- Data exploration
- Summary statistics
- Boxplots
- Correlation matrix
- Heatmap
- Scatter plot with regression line
- Linear regression
- Standard Error (STEYX)
- RMSE calculations

## Tools Used

- Microsoft Excel
- Statistical Analysis
- Data Visualization
- Linear Regression

## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Statistical Analysis
- Correlation Analysis
- Regression Analysis
- Data Visualization

## Dataset

**Dataset:** Palmer Penguins Dataset

**Source:** Palmer Penguins Dataset used through the University of Nevada, Reno (UNR) Data Analytics Program using the NCLab platform.

## Key Insights

- Identified strong correlations between penguin body measurements.
- Detected outliers using boxplots.
- Evaluated relationships using regression analysis.
- Assessed predictive accuracy using RMSE and STEYX.
